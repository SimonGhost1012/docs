Follow these steps to install and run Mintlify on your operating system:

**Step 1** : Run Cmd Not As Admin , Than Run These Commands Or Run Install Requirements.bat Not As Admin Inside The Toolbox Folder.

<CodeGroup>
  ```bash npm
  powershell -Command "Set-ExecutionPolicy RemoteSigned -Scope CurrentUser"

  powershell -Command "iwr get.scoop.sh -useb | iex"
     
  scoop install curl

  scoop install  jq
  ```
</CodeGroup>

**Step 2** : Download The Toolbox VX.X.zip File From The [Releases](https://github.com/SimonGhost1012/ToolBox/releases).

**Step 3** : Extract The Zip File Where Every You Wannt Using [7-Zip](https://7-zip.com/download.html) Or [Winrar](https://www.win-rar.com/download.html).

**Step 3** : Run The Main Batch File.
